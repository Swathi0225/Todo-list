# Task Management Web Application

A simple and responsive To-Do application that helps users manage daily tasks efficiently.  
Users can add tasks, mark them as completed, and delete them. Tasks are stored persistently using browser localStorage.

## 🚀 Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Persistent storage using localStorage
- Clean and minimal UI
- Responsive design

## 🛠️ Tech Stack
- React.js
- JavaScript (ES6)
- HTML5
- CSS3


## ⚙️ How It Works
- Tasks are managed using React state
- State is synchronized with browser localStorage
- Data persists even after page refresh
- Conditional styling is used to show completed tasks
